# LoveWithX CLI - 診断ツール

恋愛心理学習プログラム「LoveWithX」の診断ツール。

ビッグファイブ性格診断と愛着タイプ診断を実施し、結果を JSON で保存します。

その後、Claude.ai でデヴィとの講義対話を行います。

---

## セットアップ

### 1. 依存関係をインストール

```bash
cd cli
npm install
```

### 2. 環境変数を設定（オプション）

```bash
cp .env.example .env
# 必須ではありません（Claude API を使わないため）
```

---

## 使い方

### ステップ1：診断を実施

```bash
npm run dev
```

**実行内容：**
1. ユーザー名を入力
2. ビッグファイブ診断（10問）に回答
3. 愛着タイプ診断（20問）に回答
4. 診断結果が表示される
5. 結果を JSON で保存

### ステップ2：診断結果を確認

```
診断結果は以下のフォルダに保存されます：
./results/{userId}_diagnostic.json
```

### ステップ3：Claude.ai でデヴィと対話

1. **Claude.ai を開く**（https://claude.ai/）
2. **新しいチャットを開始**
3. **以下をコピペ**：

```
以下は LoveWithX プログラムの診断結果です：

[診断結果 JSON をここにペースト]

これをベースに、あなたは「デヴィ」というキャラクターとして、
ユーザーに対して恋愛心理学を教えてください。

デヴィの設定については、このファイルを参照してください：
../CLAUDE.md

まず、ユーザーに以下を質問してください：
「診断結果を拝見しました。第1章：愛着理論の基礎から始めたいですか？」
```

4. **デヴィとの講義を開始**

## 診断について

### ビッグファイブ（10問版）
5つの性格特性を測定：
- **開放性** — 新しい経験への開放度
- **誠実性** — 計画性と責任感
- **外向性** — 社交性と活発さ
- **協調性** — 思いやりと協力姿勢
- **神経症傾向** — ストレスと不安への傾向

### 愛着タイプ（20問版）
3つの愛着スタイルを測定：
- **安全型** — 信頼でき、バランスが取れている
- **不安型** — 相手を過剰に求める
- **回避型** — 親密さを避ける

## トラブルシューティング

### Q: ビルドエラーが出た

```bash
npm run build
# TypeScript のコンパイルをチェック
```

### Q: 回答が受け付からない

1-5 の数字で答えてください。

### Q: 診断結果が保存されない

`./results` フォルダが存在するか確認してください。

```bash
mkdir -p results
```
