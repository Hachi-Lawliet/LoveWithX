## 診断ラインアップ

### 1. ビッグファイブ性格診断

**目的**：学習者の基本的な性格特性を理解

**5つの特性**：
1. **開放性（Openness）** — 新しい経験・創造性への開放度
2. **誠実性（Conscientiousness）** — 計画性・責任感
3. **外向性（Extraversion）** — 社交性・活発さ
4. **協調性（Agreeableness）** — 思いやり・協力姿勢
5. **神経症傾向（Neuroticism）** — ストレス・不安への傾向

**質問形式**：
- 10問版または20問版
- 5段階評価（1:全く当てはまらない〜5:とても当てはまる）

**スコアリング**：
- 各特性ごとに平均スコアを算出
- 結果を「高い / 中程度 / 低い」で評価

**結果出力例**：
```
あなたのビッグファイブ結果：
開放性：3.2（中程度）
誠実性：4.1（高い）
外向性：2.8（低い）
協調性：4.3（高い）
神経症傾向：2.5（低い）

→ 解釈：「あなたは責任感が強く、思いやりのある性格。
　社交より深い関係を大切にする傾向があります。」
```

**恋愛への応用**：
- 開放性が低い：信頼できる相手との安定した関係を求める
- 外向性が高い：パーティーなど社交的な相手と相性が良い
- 神経症傾向が高い：不安が強いため、相手の確認が必要
- 協調性が高い：相手を優先しすぎる傾向

### 2. 愛着タイプ診断

**目的**：恋愛パターンの根底にある心理的基盤を理解

**タイプ分類**：
1. **安全型（Secure Attachment）** — 信頼でき、バランスが取れている
2. **不安型（Anxious Attachment）** — 相手を過剰に求める、見捨てられ不安
3. **回避型（Avoidant Attachment）** — 親密さを避ける、自立志向

**質問形式**：
- 15〜20問
- 各タイプの傾向を測定

**結果出力例**：
```
あなたの愛着タイプ：混合型（不安型寄り）
- 安全型：35%
- 不安型：55%
- 回避型：10%

→ 解釈：「あなたは相手との関係を大切にしたい気持ちが強い一方で、
　見捨てられることへの不安があります。」
```

**恋愛への応用**：
- 安全型：相手との関係が安定しやすい。コミュニケーションが得意
- 不安型：相手を追ってしまう傾向。自分軸を持つことが重要
- 回避型：親密さを避けるため、関係が進展しにくい傾向

### 3. 相性診断

**目的**：学習者と相手の心理的なマッチングを測定

**入力形式**：
- 学習者のビッグファイブ・愛着タイプ
- 相手についてのヒアリング（質問形式で把握）

**マッチング判定**：
- 各特性・愛着タイプの「相補性」「類似性」を分析
- スコアを算出（0〜100点）

**結果出力例**：
```
相性スコア：72点（良好）

理由：
- 開放性：あなたは保守的、相手は新しいことを好む →補完関係
- 外向性：あなたは内向的、相手は外向的 → 相手が社交をリード
- 愛着：あなたは不安型、相手は安全型 → 相手が安心感を提供

推奨：「相手のリードに身を委ねると、関係がうまくいきやすいでしょう」
```

### 4. 恋愛スタイル診断（Lee's Love Styles）

**目的**：学習者がどんな形で相手を愛するかを理解

**6つのスタイル**：
1. **Eros（エロス）** — 情熱的で肉体的な愛
2. **Ludus（ルドゥス）** — 遊びのような軽い愛
3. **Storge（ストルゲ）** — 友情に近い愛
4. **Pragma（プラグマ）** — 現実的・理性的な愛
5. **Mania（マニア）** — 執着的で不安定な愛
6. **Agape（アガペー）** — 無条件の献身的な愛

**質問形式**：
- 18問（各スタイル3問ずつ）

**結果出力例**：
```
あなたの恋愛スタイル：Storge（友情的愛） & Agape（献身的愛）
スコア分布：
- Eros：2.1 / 5
- Ludus：1.8 / 5
- Storge：4.2 / 5 ← 最高
- Pragma：3.1 / 5
- Mania：1.5 / 5
- Agape：3.9 / 5

→ 解釈：「あなたは相手との長期的な信頼関係を大切にするタイプ。
　友達のように関係が深まる愛し方が得意です。」
```

### 5. エピソード分析診断

**目的**：学習者の具体的な恋愛エピソードから、タイプ・課題を診断

**入力形式**：
- ユーザーが「昨日〜という状況でこう反応した」というエピソードを入力
- フリーテキスト or 構造化質問

**分析項目**：
- 本当の感情 vs 表現された行動の乖離
- 相手への影響（相手の心理状態の推測）
- 改善のための「菫のやり方」を提示

**結果出力例**：
```
エピソード：「相手から連絡が来ず、不安になって先に連絡してしまった」

分析：
✓ 本当の感情：見捨てられ不安（愛着：不安型の傾向）
✗ 相手への影響：相手が「追われている」と感じ、さらに距離を置く可能性

改善案（菫のやり方）：
「不安な気持ちは自然です。でも、そこでぐっと待つことが、相手が
あなたを求める状況を作ります。あなたが相手のことを思って待っている、
その時間が相手の心を動かすんです。」
```
